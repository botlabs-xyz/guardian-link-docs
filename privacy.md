## Privacy Policy

Guardian Link processes limited Discord metadata required to provide moderation alerts, logging, and security workflows.

### Data processed

- Discord server, channel, user, and role IDs
- Moderation event metadata
- Alert and logging configuration metadata
- Invite logging metadata when enabled

### Why data is processed

- Deliver alerting and moderation intelligence features
- Maintain incident logs for staff review
- Improve reliability and prevent abuse

### Data sharing

Guardian Link does not sell user data. Data may be processed by infrastructure providers needed to run the service.
